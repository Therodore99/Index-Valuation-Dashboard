# 恒生科技指数：近1年区间位置展示（V1 - iFinDPy版）

## 一、目标

用 Python 编写一个最小可运行脚本，模仿参考截图的展示形式：

以“区间最低值 — 当前值 — 区间最高值”的横向刻度方式，
展示恒生科技指数（HS Tech）当前指数值在近1年历史区间中的相对位置。

视觉效果参考：
- 一条横线表示近一年区间
- 左侧标注区间最低值
- 右侧标注区间最高值
- 中间用红点标注当前值
- 标注具体数值

目标是实现一个可运行的单指数版本（恒生科技）。

---

## 二、数据

### 指数
- 恒生科技指数（HS Tech）
- 使用 iFinDPy 获取数据（本地 SDK / 客户端环境）
- 指数代码：`HSTECH.HK`（如实际 iFinDPy 需要其他代码，以 CONFIG 可配置为准）

### 时间范围
- 近1年（从当前日期往前365天）
- 若交易日不足365天，使用实际返回交易日
- 当前值定义为：距当前日期最近一个交易日的收盘价（或最新可得值）

### 指标
- 指数收盘价（点位）
- 不使用 PE/PB，仅使用指数点位

---

## 三、数据获取方式（iFinDPy）

### 登录
脚本需要先登录 iFinD（iFinDPy）后再取数。

登录信息不得硬编码在代码中：
- 优先从环境变量读取：`IFIND_USER`、`IFIND_PASS`
- 如果读取不到，脚本应提示用户设置环境变量并退出

### 拉取方式
使用 iFinDPy 的“日期序列/历史行情”能力获取近1年每日数据（推荐日期序列接口）：

- 目标：获得一个按日期排列的时间序列（date, close）

提示：具体调用函数与字段由 CONFIG 控制区集中配置，便于后续调整：
- code（指数代码）
- indicator（收盘价指标名称，例如 `ths_close_price_index` 等）
- 起止日期格式（YYYY-MM-DD 或 YYYYMMDD）

---

## 四、数据逻辑

需要获取：

1. 近1年每日指数收盘价时间序列
2. 计算：
   - 区间最低值（min）
   - 区间最高值（max）
   - 当前值（最近交易日收盘价）
3. 计算当前值在区间中的位置比例：

position = (current - min) / (max - min)

并以百分比形式输出。
