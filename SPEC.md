# 指数区间位置展示项目（V2 - iFinDPy版）

## 一、项目目标

用 Python 编写一个可复用、可扩展、可维护的指数数据分析项目。

当前阶段目标是：
以恒生科技指数为示例，展示多个指标在指定时间范围内的历史区间位置。

当前支持的指标包括：
- 指数收盘价（close）
- 市盈率（pe）
- 市净率（pb）

展示方式参考目标样式：
- 一条横向区间线表示历史区间
- 左侧标注区间最低值
- 右侧标注区间最高值
- 中间用红点标注当前值
- 显示当前值在区间中的相对位置

本项目后续会扩展到更多指数，因此代码必须具备良好的可复用性。
新增其他指数时，应尽量只修改配置，而不需要重写主流程代码。

---

## 二、数据来源

- 使用 iFinDPy 获取数据
- 不使用 iFinD HTTP API
- 本地环境已具备 iFinDPy 使用条件

### 登录要求
登录信息不得硬编码在代码中：
- 优先从环境变量读取：
  - IFIND_USER
  - IFIND_PASS
- 如果读取不到，程序应提示用户设置环境变量并退出

## 三、指数与指标设计

## 1. 指数配置
代码中应采用统一配置方式管理指数信息，例如：

- 指数名称
- 指数代码
- 各指标对应的 iFinD 指标名称

当前默认先实现：
- 恒生科技指数
- 指数代码：HSTECH.HK

后续应方便扩展到：
- 上证50
- 沪深300
- 中证500
- 创业板指
- 其他宽基指数

## 2. 指标配置
当前支持以下指标：

- close：指数收盘价
- pe：市盈率
- pb：市净率

代码中不得把指标逻辑写死为单一指标，而应通过配置统一处理。

## 四、时间范围

- 当前默认支持近1年区间展示
- 时间范围计算方式：从当前日期往前365天
- 若交易日不足365天，则以实际返回交易日为准
- 当前值定义为：距离当前日期最近一个交易日的数值

代码设计上应为后续扩展到近5年保留能力。

---

## 五、数据缓存与复用

为了减少 iFinD 数据消耗，必须实现本地 CSV 缓存机制。

## 1. 缓存规则
每个指数、每个指标分别保存一个 CSV 文件。

推荐目录结构：

data/
  hstech/
    close.csv
    pe.csv
    pb.csv

## 2. CSV要求
每个 CSV 至少包含两列：
- date
- value

## 3. 增量更新逻辑
脚本运行时应：

1. 先检查本地是否已有对应 CSV
2. 如果已有：
   - 读取已有历史数据
   - 判断最新日期是否已覆盖到最近交易日
   - 若未覆盖，则仅从 iFinD 获取缺失日期的数据
   - 将新旧数据合并、去重、按日期排序后重新保存
3. 如果没有：
   - 从 iFinD 拉取完整区间数据并保存 CSV

## 4. 目标
例如当需要近5年数据时：
- 不重复获取历史已有数据
- 只补充最新缺失部分
- 降低 iFinD 调用消耗

---

## 六、数据逻辑

对于每个指数、每个指标，需要获取一段时间内的时间序列数据，并计算：

- 区间最低值 min_value
- 区间最高值 max_value
- 当前值 current_value（最近交易日）
- 区间位置比例 position_pct

计算公式：

position_pct = (current_value - min_value) / (max_value - min_value) * 100

当 max_value == min_value 时，需要做保护处理，避免除零错误。

区间最低值: xxxx.xx

## 十、模块职责要求

### main.py
作为项目入口文件，只负责流程调度，不写具体业务细节。

流程包括：
- 登录 iFinD
- 按配置遍历指数与指标
- 获取数据
- 更新缓存
- 计算结果
- 输出命令行结果
- 生成图表

### config.py
集中管理配置，包括：
- 指数名称
- 指数代码
- 指标配置
- 路径配置
- 时间范围配置

后续新增指数时，尽量只改这里。

### core/ifind_client.py
负责：
- iFinD 登录
- 调用 iFinD 获取原始数据
- 返回统一格式的 DataFrame

### core/data_manager.py
负责：
- 读取 CSV 缓存
- 判断是否需要补充新数据
- 增量拉取缺失数据
- 合并、去重、排序并保存

### core/calculator.py
负责：
- 计算 min / max / current / position_pct

### core/plotter.py
负责：
- 生成横向区间位置图
- 保存图像文件

### utils/date_utils.py
负责：
- 日期范围计算
- 最近日期处理

### utils/file_utils.py
负责：
- 文件夹创建
- 路径管理等辅助逻辑
